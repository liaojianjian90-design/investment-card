# 投资健康卡 5.3.4

**趋势优先 + AI新增资金优先执行版**

5.3.4 的核心变化：CRCL 加入加密金融基础设施卫星仓，同时核心增长层继续采用“总量优先判断”。BTC/ETH 合计达到 8%-13% 区间后，不再因为 BTC 或 ETH 单项偏低显示“核心不足”，而是显示“核心偏科”；该提示不影响 AI 主攻仓新增资金优先级。

## 一句话原则

```text
平时少动；上涨不追；AI 急跌敢买；BTC/XAUT 只保留底仓；新增资金优先主升方向；单笔有效；仓位达标就停。
```

## 唯一仓位来源

系统只读取：

```text
config/holdings.json
```

不使用前端手动买卖、不使用 Cloudflare 云端同步、不使用 manual-trades.json，也不使用 localStorage 覆盖仓位。

流程：

```text
交易所买入/卖出
→ 手动修改 config/holdings.json
→ 提交 GitHub
→ GitHub Actions 生成 snapshot / alerts
→ 网页和邮件按 5.3.4 规则更新
```

## 当前阶段仓位目标

| 层级 | 目标 | 说明 |
|---|---:|---|
| 现金 | 35%-45% | 保留防守和二次进攻权，但不长期 70%+ 空仓 |
| AI 主攻仓 | 25%-35% | 当前收益主线，35% 后停止新增，40% 硬上限 |
| BTC | 5%-8% | 加密底仓，不再机械补到 10% |
| ETH | 3%-5% | 加密底仓，深跌才补 |
| VOO | 5%-7% | 长期稳定底仓，优先级低于 AI A/S 机会 |
| XAUT | 3%-4% | 保险底仓，3900 上方不补，3850 以下才补 |
| DOGE | 3%-4% | BTC 企稳后的高弹性卫星仓，硬上限 5.5% |

## 新增资金优先级

```text
1. AI A/S 级急跌机会
2. MU / DRAM / GLW / SMH 正常有效回调
3. SMH 半导体篮子
4. ETH 深跌价位
5. VOO 大跌底仓
6. BTC / XAUT 仅深跌或趋势重新确认
7. DOGE 仅 BTC 企稳后小仓
```

## AI 主攻仓

第一主攻层：

```text
MU / DRAM / GLW / SMH
```

目标：

| 标的 | 目标仓位 | 停止新增 | 再平衡 |
|---|---:|---:|---:|
| MU | 4%-8% | 9% | 11% |
| DRAM / rDRAM | 4%-8% | 9% | 11% |
| GLW | 4%-8% | 9% | 11% |
| SMH | 4%-8% | 9% | 11% |

AI 总仓：25%-35%；35% 后停止新增；40% 为硬上限。

## AI 急跌机会规则

触发前提：

```text
现金 ≥ 45%
AI 主攻仓 < 25%
数据未过期
没有财报暴雷或基本面恶化
触发标的是 MU / DRAM / GLW / SMH
```

A 级机会：

```text
任一主攻标的跌幅 ≥ 5%
或两个以上主攻标的同时跌幅 ≥ 4%
```

执行金额：1000-1500 USDT。

S 级机会：

```text
现金 ≥ 55%
AI 主攻仓 < 20%
三个以上主攻标的同时急跌
SMH 跌幅 ≥ 3%
```

执行金额：1500-2500 USDT。

风控：单日 AI 主攻买入不超过总资产 12%；单一主攻标的单日通常不超过 1000 USDT；2x DRAM/RAM 单日不超过 500 USDT。

## BTC / ETH 新规则

BTC 不再在 58000 / 57000 自动提醒买入。

| 条件 | 动作 |
|---|---:|
| BTC 58k-60k 弱势震荡 | 不买 |
| BTC ≤ 56,000 且人工确认止跌 | BTC 500 + ETH 250 |
| BTC ≤ 55,000 且无系统性风险 | BTC 700 + ETH 350 |
| BTC < 52,000 | 暂停 48-72 小时，不接飞刀 |
| BTC 重新站稳 62,000 | 只做趋势确认观察，不追大仓 |
| ETH ≤ 1,500 | 买 500 |
| ETH ≤ 1,350 | 买 800 |

## XAUT / VOO 新规则

XAUT 已经有第一笔底仓后，不再因为 3950-3980 小波动继续补。

| 条件 | 动作 |
|---|---:|
| XAUT 3900 上方 | 持有，不补 |
| XAUT 3920-3900 | 默认暂缓；若 AI 没有 A/S 机会，最多 300 |
| XAUT ≤ 3850 | 买 500 |
| XAUT 仓位 ≥ 4% | 停止新增 |
| VOO 大跌至预设价位 | 可补底仓，但优先级低于 AI A/S 机会 |

## DOGE 规则

DOGE 可以作为 BTC 上行周期放大器，但必须满足：

```text
BTC 已经企稳或重新走强
DOGE+BGB 合计低于 4.5%
现金仍高于 40%
不是 BTC 下跌途中盲目补 DOGE
```

DOGE >5.5% 时卖出 25% 回现金；DOGE+BGB >6% 为高风险警戒。
